# smol-rag

## Project Stack Overview

This project uses the following stack for building a Retrieval-Augmented Generation (RAG) system. The components are placeholders and can be updated as the project evolves:

1. **LLM Inference Layer**: e.g., Ollama (to be finalized)
2. **LLM Model**: e.g., Mistral 7B (to be finalized)
3. **Embeddings**: e.g., nomic-embed-text (to be finalized)
4. **Vector Database**: e.g., Chroma (to be finalized)
5. **Framework Layer**: e.g., LangChain (to be finalized)

### Notes
- Each component can be replaced or updated based on project requirements.
- Ensure compatibility between the LLM model, embeddings, and vector database.
